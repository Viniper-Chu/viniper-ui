## v0.2.3 (2026-05-17)

### 修复
- **左上角图标**：将狗头图标替换为新图标，统一桌面快捷方式和网页 favicon
- **新建会话命名**：新建会话时若未输入名称，自动命名为"新建会话（1）（2）（3）…"
- **删除会话**：修复删除会话按钮无响应的问题，增加错误提示和事件隔离
- **记忆隔离**：每个会话使用独立的记忆目录，不再共享 Claude Code 记忆
- **权限确认**：权限确认改为执行中按需询问，不再提前预判拦截

### 技术细节
- server.py: 新增 `re.match` 自动编号逻辑，`DATA_DIR/session-memory/` 目录用于会话记忆隔离
- app.js: `needsPermissionForPrompt` 简化，删除按钮增加 `stopPropagation` 和错误处理
- 图标资源统一使用 `claude.ico`
